# CineLux • Cartelera (HTML + CSS + JS)

Proyecto listo para tu curso: una cartelera de cine con diseño moderno.

## Cómo correrlo (Visual Studio Code)

Opción A (recomendada): **Live Server**
1. Abre la carpeta `cine-cartelera` en VS Code.
2. Instala la extensión **Live Server**.
3. Click derecho a `index.html` → **Open with Live Server**.

Opción B: abrir el archivo directamente
- Doble click a `index.html`.

## Qué incluye
- Búsqueda por título
- Filtros por género
- Orden (popularidad / rating / título / duración)
- Modal de detalles
- Selector simple de asientos
- “Mis tickets” guardado en `localStorage`
- Tema oscuro / claro

## Personalización rápida
Edita el dataset en `app.js` (const `MOVIES`).

Hecho sin frameworks.
