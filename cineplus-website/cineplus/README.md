# CinePlus - Cartelera de Cine (HTML + CSS + JS)

Proyecto de curso: sitio web de cine con varias páginas, banners, filtros, modal, selección de asientos y tickets guardados con LocalStorage.

## Páginas
- `index.html` (Inicio con carrusel + destacados)
- `pages/cartelera.html` (Listado + filtro + búsqueda)
- `pages/pelicula.html` (Detalle por película)
- `pages/asientos.html` (Seleccionar asientos y comprar)
- `pages/estrenos.html` (Próximos estrenos)
- `pages/tickets.html` (Historial de compras)
- `pages/contacto.html` (Formulario con validación)
- `pages/faq.html` (Preguntas frecuentes)
- `pages/nosotros.html` (Sobre el cine)

## Cómo correr
1) Abre la carpeta en VS Code
2) Recomendado: usa la extensión **Live Server**
3) Abre `index.html`

## Nota
Todo funciona sin backend. Los datos de películas están dentro de `assets/js/app.js`.
